# Project Setup Manifest
*Read this first. Do this once after Week 3 is complete.*

---

## Files to Upload to the Project

| File | Purpose | Update Frequency |
|---|---|---|
| `coaching-master.md` | Athlete profile, injury, equipment, nutrition, daily work, pipeline | Rarely — only when something fundamental changes |
| `iron-tide-active.md` | Current program targets, session log, active coaching calls | End of each week |

That's it. Two files. Everything else lives in those two documents.

---

## Step-by-Step Project Setup

**1. Create a new Project in Claude.ai**
- Name it something like "KB Coaching"

**2. Add the system prompt**
- Open `project-system-prompt.md`
- Copy the entire contents
- Paste into the Project Instructions field

**3. Upload both files**
- Upload `coaching-master.md`
- Upload `iron-tide-active.md`

**4. Start your first session**
- Open a new conversation inside the Project
- Drop your session screenshot
- Type "W4D1" (or whatever session it is)
- Claude will read both files and respond with targets

---

## Weekly Maintenance (2 minutes)

At the end of each week after your last session:

1. Ask Claude: *"Generate the updated iron-tide-active.md for this week"*
2. Claude will output the full updated file
3. Download it
4. In the Project, delete the old `iron-tide-active.md`
5. Upload the new one

That's the only ongoing maintenance required.

---

## Program Transitions

When Iron Tide ends and you move to ABF:

1. Ask Claude to generate `abf-active.md`
2. Archive `iron-tide-active.md` somewhere locally (it's your training record)
3. Upload `abf-active.md` to the Project
4. Update `coaching-master.md` if anything has changed (equipment, injury status, etc)
